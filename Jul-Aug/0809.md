## 2164 카드 2

```python
# 큐 연습문제

from collections import deque

n = int(input())
a = [i+1 for i in range(n)]
s = deque(a)

while len(s) > 1:
    s.remove(s[0])
    s.append(s.popleft())
    
print(*s)
```

## 11866 요세푸스 문제 0 

```python
# 큐 연습문제

from collections import deque

n, k = map(int, input().split())
a = [i+1 for i in range(n)]
s = deque(a)

result = []

while len(s) > 0:
    for i in range(k-1):
        s.append(s.popleft())
    result.append(s.popleft())
    
print('<', end='')
print(*result, sep=', ', end='')
print('>')

```
